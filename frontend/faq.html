Frequently Asked Questions

How do I use this application?

To use this application, simply enter the ingredients you want to use into the search bar and click the "Search" button. The application will fetch and display recipes that you can make with those ingredients. If there are no exact matches for the ingredients you entered, the application will display recipes that contain similar ingredients.

Where does the recipe data come from?

The recipe data is fetched from a backend API that provides a list of recipes based on the ingredients you enter. The API data is
from this Kaggle dataset.

Can I save recipes for later?

Currently, the application does not have a feature to save recipes for later.

How can I contact the developer?

You can contact the developer via GitHub to request changes or report bugs.
